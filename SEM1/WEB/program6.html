CSS Tutorial

Inline
Inline CSS:
Inline CSS refers to the method of applying styles directly to a specific HTML element using the style attribute within the element's opening tag.

Key Characteristics:
  • Direct Application: Styles are applied directly to the HTML element.
  • High Specificity: Inline styles override other styles.
  • Limited Reusability: Inline styles cannot be reused on other elements.
Syntax: <div style="color: red;">Content</div>
Internal CSS:
Internal CSS is defined in the <style> element inside the <head> section of an HTML document. It allows you to style a single page without affecting others.

Key Characteristics:
  • Scoped to the Page: Styles apply only to elements within that page.
  • Organized: All styles are within a single document.
  • Can be easily modified: Unlike inline CSS, changes can be made in one place.
Syntax: <style> p { color: blue; } </style>
External CSS:
External CSS is defined in a separate CSS file, which is linked to an HTML document. This is the most efficient method for styling websites that have multiple pages.

Key Characteristics:
  • Separation of Concerns: HTML and CSS are kept separate, improving maintainability.
  • Reusable: A single CSS file can be used across multiple pages.
  • Faster Load Times: Once downloaded, the CSS file is cached for faster page loads on subsequent visits.
How to implement:

1. Create a CSS file with a .css extension (e.g., styles.css)
2. Link it to your HTML with the <link> tag: <link rel="stylesheet" href="styles.css">
